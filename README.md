# 🏥 Multilingual AI Medical Assistant

A **Flask web app** that uses **Google Gemini AI** and **Deep Translator** to provide **basic medical assessments** in **multiple Indian languages** 🇮🇳.  
Enter symptoms → Get a simple health analysis → In your preferred language!


## ✨ Features

- 🔥 Symptom analysis using **Gemini 1.5 Pro**
- 🌐 Supports 12+ Indian languages (Hindi, Bengali, Tamil, Telugu, Marathi, etc.)
- 🔄 Auto translation of input/output
- 🧠 AI-generated response includes:
  - Possible conditions
  - Severity (Low, Medium, High)
  - Next steps (home care, doctor consultation, emergency)
  - General care suggestions
- 🛡️ Standard disclaimers added automatically
- 🖥️ Clean Flask backend ready for frontend integration
